# RB-008: Prompt Injection or Content Firewall Breach Attempt

## Trigger
- Firewall L2/L3 blocks surge, or known injection signatures bypass L2.
- Suspicious tool-call proposals reaching execution gate.

## Detection
- Inspect `content_firewall_logs` and execution decisions around incident window.
- Check semantic verifier failures for malformed outputs.

## Immediate Actions
1. Force high-risk tools to `REQUIRE_APPROVAL` mode.
2. Increase firewall strictness for affected workspace/channel.
3. Quarantine suspicious ingress payloads for forensic analysis.

## Mitigation
- Add new attack patterns to L2 and update semantic classifier features.
- Tighten schema firewall for targeted tool inputs.
- Validate recipient verification and memory write gates.

## Recovery
- Re-enable normal policy only after red-team replay passes.

## Verification
- Injection suite passes; blocked attempts are logged with evidence.
- No unauthorized commit-phase tool execution occurred.

## Escalation
- Security incident process if bypass resulted in external side effects.
