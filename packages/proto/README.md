# @brevio/proto

Protobuf contracts for Brevio inter-service gRPC communication.

## Structure
- `brevio/common/v1` shared messages
- `brevio/gateway/v1` gateway service contract
- `brevio/brain/v1` brain service contract
- `brevio/hands/v1` hands service contract
- `brevio/auth/v1` auth service contract
- `brevio/profile/v1` profile service contract
- `brevio/scheduler/v1` scheduler service contract
- `brevio/edge/v1` edge relay contract
