# morning-manifesto

Brain-plane reflection and action framing skill for daily executive readiness.

## Supported actions

- `generate_manifesto`: produces a morning mindset brief with prioritized actions.
- `sync_actions`: prepares actionable outputs for downstream tools (Reminders, Linear, Obsidian).

## Notes

- Runs deterministic local logic without external APIs.
- Pairs naturally with `plan-my-day` and `daily-rhythm` outputs.
