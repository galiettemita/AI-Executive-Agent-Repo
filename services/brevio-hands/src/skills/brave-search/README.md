# brave-search

Privacy-oriented web search adapter.

- Plane: `hands`
- External API target: Brave Search API
- Auth: API key

## Input

- `query` (required)
- `max_results` (optional)

## Output

- `provider`: `brave-search`
- `results[]`

## Brevio use case

"Search this topic with privacy-first results" for alternate provider routing.
