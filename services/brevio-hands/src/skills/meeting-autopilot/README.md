# meeting-autopilot

Brain-plane meeting synthesis skill for summaries, decisions, actions, and follow-up drafts.

## Supported actions

- `summarize_meeting`: condenses transcript into concise meeting summary.
- `extract_actions`: extracts actionable owner-task pairs.
- `draft_follow_up`: generates a structured follow-up email draft.

## Notes

- Requires transcript input for all actions.
- Deterministic parsing and extraction path (no external API dependency).
