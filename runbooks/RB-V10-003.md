# RB-V10-003: Billing, Subscriptions, and Cost Management

## Scope
Cost rollup, subscription lifecycle, billing enforcement, budget alerts.

## Key Tables
- `cost_rollups` — aggregated workspace costs
- `revenue_events` — revenue tracking
- `admin budget` — monthly caps per workspace

## Operations

### Check Workspace Budget
```bash
curl https://admin.brevio.app/v1/admin/costs/budgets \
  -H "X-User-Role: admin"
```

### View Cost Summary
```bash
curl https://admin.brevio.app/v1/admin/costs/summary \
  -H "X-User-Role: admin"
```

### Check Cost Anomalies
```bash
curl https://admin.brevio.app/v1/admin/costs/anomalies \
  -H "X-User-Role: admin"
```

## Incident: Budget Exceeded

1. Check current spend vs cap in admin dashboard.
2. If legitimate, increase cap via admin API or direct DB update.
3. If abuse, activate kill switch (see RB-V10-001).

## Incident: Billing Webhook Failures

1. Check Stripe webhook logs in Stripe dashboard.
2. Verify `STRIPE_WEBHOOK_SECRET` env var is set correctly.
3. Check gateway service logs for webhook processing errors.
4. Replay failed webhooks from Stripe dashboard.

## Rollback
Budget enforcement is soft — services degrade gracefully when budget is exceeded.
To disable enforcement temporarily, set workspace budget cap to a very high value.
