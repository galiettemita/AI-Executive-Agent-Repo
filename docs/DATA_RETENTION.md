# Data Retention Policy

## Categories
- PII (user identifiers, payment data): retain while account active; delete within 30 days of closure
- Conversations and logs: retain for 24 months; allow opt-out
- Voice recordings: retain for 90 days by default
- Financial data: retain while account active; delete within 30 days

## Deletion
- User-initiated deletion triggers all associated data removal
- GDPR/CCPA workflows documented and tracked
