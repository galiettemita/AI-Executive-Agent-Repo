# perplexity

Web-grounded answer generation with citations.

- Plane: `hands`
- External API target: Perplexity API
- Auth: API key

## Input

- `query` (required)
- `model` (optional)

## Output

- `provider`: `perplexity`
- `answer`
- `citations[]`

## Brevio use case

"Give me a cited summary of recent developments" for research-grade responses.
