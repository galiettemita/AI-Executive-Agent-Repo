# Access Review Runbook

## Frequency
- Quarterly for all admin accounts and service accounts.

## Steps
1. List all admins in GitHub, Render, Meta, Stripe, Google, Microsoft, and any cloud providers.
2. Remove accounts no longer required.
3. Verify MFA enabled for all admins.
4. Document findings and store evidence in SOC2 tracker.

## Evidence
- Screenshot of each admin list
- Notes on removals and approvals
